# UNI-PASS-Customs-Library
[![UNI-PASS-Customs-Library](./public/screenshot.png)]
## 어떤 프로그램인가요?
해외직구 통관정보조회 프로그램입니다.   
수입화물진행정보(건별) 조회는 구현 하지 않았습니다.   

## 사용 방법은 무엇인가요?
개인통관번호, 날짜 입력하시면 됩니다.   

## 데이터는 어디서 가져오나요?
[UNI-PASS](https://unipass.customs.go.kr/) 에서 가져 옵니다.
